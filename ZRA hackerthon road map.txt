## *Ezra — GhostBuster Module (Phantom Detection)*

*Task 1 — Environment Setup*

- Install Python, Pandas, SQL libraries.
- Set up Git repo + branching strategy.
- Create virtual environment.
- Document setup instructions.

*Task 2 — Data Preparation*

- Collect payroll, PACRA, NAPSA, NRC mock data.
- Standardize into CSV/JSON schemas.
- Anonymize sensitive fields.
- Store datasets in /data/ folder.

*Task 3 — Detection Logic*

- Implement missing NRC/ID detection.
- Detect duplicate IDs across companies.

*Task 4 — API Service*

- Build Flask/FastAPI service.
- Endpoint: POST /ghost-check (upload dataset).
- Endpoint: GET /ghost-report/:company_id.
- Output JSON + CSV reports.

*Task 5 — Testing*

- Write unit tests for matching logic.
- Run API integration tests.
- Benchmark detection on large dataset.
- Validate against sample anomalies.

*Task 6 — Documentation*

- API usage guide (inputs/outputs).
- Detection rulebook.
- Test cases + results.
- Integration notes for dashboard (Thomas).

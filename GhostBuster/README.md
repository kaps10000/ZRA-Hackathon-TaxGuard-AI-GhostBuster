# GhostBuster - Ghost Employee Detection System

**Part of: ZRA Hackathon - TaxGuard AI (Ezra Branch)**

A state-of-the-art system to detect ghost employees in the Zambian government using multi-source data analysis.

## Quick Start

```bash
cd GhostBuster/backend
pip install -r requirements.txt
python generate_datasets.py
python app.py

# New terminal
cd GhostBuster/frontend
npm install
npm start
```

Access at: http://localhost:3000

## Features

- Multi-source cross-referencing (NAPSA, Home Affairs, Banks)
- Bank withdrawal pattern analysis (UNIQUE!)
- Evidence-based detection with severity levels
- Individual & batch processing
- Export to CSV/JSON

## Impact

**Test Data:** 10,000 employees
**Ghost Detected:** 3,000 (30%)
**Annual Savings:** K360,000,000

# Dashboard Frontend

This folder contains the React dashboard app for investigators and ZRA.

Quick start (dev machine):

1. cd frontend
2. npm install
3. npm start

Structure:

- `src/` - React source files
- `public/` - Static assets
- `components/` - Reusable React components
- `charts/` - Chart components
- `pages/` - Dashboard pages

Integration:
- The frontend expects APIs from AI Risk, GhostBuster, WhistlePro, and Predictive modules. For now, use mocked endpoints in `api_integration/`.
